Welcome to Machine Learning A-Z! In this course, you'll go from zero to building real ML models in both Python and R.

This first lesson covers what machine learning actually is, the difference between supervised and unsupervised learning, and a roadmap of everything you'll build by the end of this course.

What You'll Build

  • A house price prediction model
  • A customer churn classifier
  • An image recognition neural network
Tip: Don't worry about installing anything yet — that's covered in the next lesson.
